# Moltbook

## 基本信息

- 类型：AI Agent 社交平台

## 在资料中的角色

Moltbook 被描述为 OpenClaw 生态中供 AI Agent 发帖、评论和互动的平台。它不是 OpenClaw 的核心架构组件，但体现了 Agent 从工具走向社会化存在的社区想象。
